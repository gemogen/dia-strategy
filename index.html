Три стратегических направления · 2026–2027

01
ИИ-развитие
Системная ИИ-инфраструктура дирекции — база знаний, инструменты качества и накопление экспертизы, которое растёт с каждым отчётом
  • База знаний (4 хранилища) — правила, корпус, телеметрия, исходы · оркестратор для автозахвата
  • LLM-as-judge — автоматическая оценка каждого отчёта по критериям: простота, обоснованность, тон, формат
  • Методолог / ИИ-специалист — выделенная роль: промпты, обратная связь, контроль дрейфа качества
  • Интеграция источников — Rocket.Chat API, транскрипции звонков с ролями, earnings calls из Bloomberg
  • Уровень 2 обратной связи — ИИ предлагает изменения промптов, методолог утверждает

02
Аналитика на базе ИИ
Официальный проект с выделенным ресурсом — покрыть 500 бумаг S&P 500 ИИ-контентом на базе консенсуса до конца 2026 года
  • ИИ-фабрика (400 бумаг) — консенсус Bloomberg + ИИ-генерация · BUY / HOLD / SELL / AVOID
  • Глубокое покрытие (100 бумаг) — проприетарная модель · тот же движок генерации
  • Скорость ×10 — 2 часа на правку черновика вместо 1–2 недель · тот же штат (5 + 1)
  • RU + EN — контролируемый перевод · числа не меняются · полный аудит-трейл
  • Все каналы сбыта — BDM · КБО · советники · приложение · рассылки

03
Многофакторный продукт
Дополнительный пласт аналитики — факторный скоринг акций поверх рекомендаций · «аудитор, не судья»
  • Факторное ранжирование — value · quality · momentum · earnings revision · каждый фактор с весом
  • Аудитор рекомендации — усиливает аналитика, не заменяет · флагирует расхождения между фундаменталом и скорингом
  • Модельные портфели — формирование на основе факторной модели · оценка риска/доходности
  • Простота для розницы — понятный скоринг поверх рекомендации · аналог quant watchlist (Motilal Oswal, DB)
  • Фаза 1 — Excel / Python прототип · co-design с аналитиками · простая линейная/lasso-модель
