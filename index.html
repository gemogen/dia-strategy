F/N
Факты и нарратив разделены: числа берутся только из верифицированного источника, ИИ формирует лишь текст вокруг готовых фактов — модель не придумывает цифры
A/H
Защита от галлюцинаций: описания компаний из проверенного словаря, мультипликаторы — из данных. Модель не генерирует факты, которые может выдумать
L2
Обратная связь второго уровня: редактор тегирует правки → Claude предлагает улучшение промптов → методолог утверждает. Система учится на ошибках
D1
Измерение с первого дня: точность идей, доля правок, время на проверку фиксируются сразу — ретроспективно KPI-базу не восстановить
